Does sklearn work?¶

sklearn is one of the favorite predictive modeling libraries. Does it work? Let's try a decision boundary plot with nearest neighbors method

You should see a yellow/purple figure with yellow and purple dots. (This is 1-nearest-neighbors' decision boundary plot)

In [20]:
N = 100
X1 = np.random.normal(size=N)
X2 = np.random.normal(size=N)
Y = X1 + X2 + np.random.normal(size=N) > 0
X = np.stack((X1, X2), axis=1)
m = KNeighborsClassifier(n_neighbors=1).fit(X, Y)
## Now greate the regular grid
ex1 = np.linspace(X1.min(), X1.max(), 100)
ex2 = np.linspace(X2.min(), X2.max(), 100)
xx1, xx2 = np.meshgrid(ex1, ex2)
                           # unlike R-s 'expand.grid', meshgrid creates matrices
g = np.stack((xx1.ravel(), xx2.ravel()), axis=1)
                           # we create the design matrix by stacking the xx1, xx2
                           # after unraveling those into columns
## predict on the grid
hatY = m.predict(g).reshape(100,100)
                           # imshow wants a matrix, so we reshape the predicted
                           # vector into one
plt.figure(figsize=(10,10))
ax = plt.subplot(1,1,1)
ax.set_aspect('equal')
ax.imshow(hatY, extent=(X1.min(), X1.max(), X2.min(), X2.max()),
           interpolation='none', origin='lower',
                           # you need to specify that the image begins from below,
                           # not above, otherwise it will be flipped around
           alpha=0.3)
ax.scatter(X1, X2, c=Y, edgecolor='k')
plt.show()

n = 1200  # how many points
x1 = np.random.normal(size=n)
x2 = np.random.normal(size=n)
alpha = np.arctan2(x2, x1)
r = np.sqrt(x1**2 + x2**2)
c1 = (np.sin(2*alpha + 2*r) > 0).astype('int')
c2 = (np.cos(2*alpha + 2*r) > 0).astype('int')
c = c1 + c2  # this is your target
# features, target:
X = np.column_stack((x1,x2))
y = c
# this is how the data looks like:
plt.figure(figsize=(7,7))
plt.scatter(x1, x2, c=c, s=40, edgecolors='k', alpha=0.6)
plt.show()
